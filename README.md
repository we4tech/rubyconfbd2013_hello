# Rubyconfbd2013Hello

This gem is created for demonstrating gem building process. This is part of RubyConfBD 2013. 

## Installation

Add this line to your application's Gemfile:

    gem 'rubyconfbd2013_hello'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rubyconfbd2013_hello

## Usage

Know the current RubyConfBd year - 

    Object.rubyconfbd_year

Know the current RubyConfBd location - 

    Object.rubyconfbd_location
